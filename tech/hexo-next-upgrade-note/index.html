更新 Next

想要更新Next时，发现已经有了天翻地覆的变化，连github的项目目录都变了，该项目已经社区化：

  • 旧目录 ~ v5.x：https://github.com/iissnan/hexo-theme-next

  • 新目录 v6.x ~：https://github.com/theme-next/hexo-theme-next

  • 删除旧的 next

  • 克隆最新的 next
